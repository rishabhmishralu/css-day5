4. What are CSS combinators

A combinator is something that explains the relationship between the selectors and element /tag.

There are four different combinators in CSS:
  1. descendant selector (space)
    2. The descendant selector matches all elements that are descendants of a specified element.
  2. child selector (>)
    3. The child selector selects all elements that are the children of a specified element.
  3. adjacent sibling selector (+)
    4. The adjacent sibling selector selects all elements that are the adjacent siblings of a specified element.
  4. general sibling selector (~)
    5. The general sibling selector selects all elements that are siblings of a specified element
